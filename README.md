# 📘 AI Engineer Handbook

Welcome to my **AI Engineer Handbook**.

This repository documents my journey toward becoming an **AI/ML Engineer**. Instead of simply using AI libraries, I focus on understanding the concepts, mathematics, and algorithms that power modern Artificial Intelligence.

The goal of this handbook is to organize my learning into a structured knowledge base that I can revisit throughout my Master's in Artificial Intelligence and beyond.

For every topic, I follow the same learning process:

```text
Theory
    ↓
Mathematics
    ↓
Intuition
    ↓
Implementation
    ↓

    ↓

    ↓
Interview Preparation
```

The implementations are maintained in separate repositories, while this handbook focuses on concepts, mathematics, intuition, and revision.
